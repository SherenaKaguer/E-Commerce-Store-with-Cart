# E-Commerce Store with Cart
A functional online shop built with pure PHP, MySQLi, HTML, and CSS.

## Features
- Session-based shopping cart
- Order management
- Product categories
- User reviews
- Admin inventory management

## Setup
1. Import `schema.sql` into your MySQL database.
2. Update `db.php` with your database credentials.
3. Start a PHP server and navigate to `login.php`.
